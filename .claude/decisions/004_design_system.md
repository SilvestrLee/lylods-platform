# Design System

Status: Accepted

Design stack:

- UI/UX Pro Max
- 21st.dev Magic
- Laravel Blade
- TailwindCSS
- AlpineJS

Design principles:

- Enterprise SaaS feel
- Clear hierarchy
- Consistent spacing
- Accessible contrast
- Visible focus states
- 44px minimum touch targets
- Predictable navigation
- Clean tables
- Helpful empty states
- Loading feedback where needed
- Semantic design tokens instead of scattered raw hex values

Current brand palette:

- Navy: #07172f
- Gold: #c9a24d
- Blue accent: #123f8c
- Warm background: #f7f3ea

Forbidden:

- React
- NextJS
- Page builders
- WordPress-style editing
- Random design drift
- Keryon design assumptions
