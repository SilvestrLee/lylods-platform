# AI Development Guardrails

Status: Accepted

This project must not drift.

The Lylods Group platform includes:

- Public website
- Backend CMS
- Admin dashboard
- Investor/user dashboard
- Investment records
- Investor notices
- Structured management features

AI assistants must not:

- Reduce the project to a brochure website
- Remove backend expectations
- Remove investor dashboard expectations
- Introduce Keryon/church/ministry assumptions
- Introduce page-builder behaviour
- Rename routes without approval
- Modify migrations without approval
- Change business logic unless fixing a clear bug
- Remove working features
- Mix admin and investor access

AI assistants may:

- Improve UI/UX
- Improve Blade structure
- Improve dashboard consistency
- Improve CMS structure
- Improve accessibility
- Improve code organization
- Improve Tailwind consistency
- Improve mobile responsiveness
