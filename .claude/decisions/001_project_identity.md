# Project Identity

Status: Accepted

Project: The Lylods Group Laravel Platform

This project is a Laravel-based business platform for The Lylods Group.

It is not Keryon.
It is not a church SaaS.
It is not Faithflow.
It is not a congregation management system.

Core identity:

- Multidisciplinary consulting organisation
- Engineering and infrastructure solutions
- Technology and innovation services
- Project management and delivery
- Training and capacity development
- Business consulting
- Recruitment and workforce solutions
- Compliance, governance, and information security services
- Investment/investor portal direction

Core modules:

- Public website
- Backend CMS
- Admin dashboard
- Investor/user dashboard
- Investment records
- Investor notices
- Contact/inquiry management

Explicitly prohibited assumptions:

- Church
- Congregation
- Ministry
- Sermon
- Pastor workflow
- Faithflow
- Keryon
- Member management
- Worship/event ministry logic
